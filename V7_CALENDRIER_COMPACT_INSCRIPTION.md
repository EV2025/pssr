# V7 — Calendrier compact et inscription simplifiée

Cette version conserve le site existant, le design global, Firebase et le cycle interactif PSSR.

## Changements principaux

- Menu principal court conservé : Accueil, PSSR, Activités & tarifs, Tableau de bord, Réserver.
- Bloc “Calendrier des activités” réorganisé en cartes compactes par jour.
- Activités affichées seulement avec : activité, horaire, âge, action.
- Formulaire d’inscription simplifié : nom, email, téléphone, module, âge si nécessaire, message facultatif, consentement et mot de passe espace membre.
- Footer professionnel conservé avec liens secondaires et informations légales.
- Le JavaScript du cycle PSSR n’a pas été modifié.
