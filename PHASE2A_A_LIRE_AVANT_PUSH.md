# À lire avant le Push Phase 2A

Cette version ajoute la Phase 2A sans remplacer la plateforme par React/FastAPI/MongoDB.

## Ne pas oublier après le Push

1. Firebase → Firestore Database → Règles : coller `firestore.rules`, puis publier.
2. Admin : ouvrir `https://ev2025.github.io/pssr/admin/`.
3. Cliquer sur `Importer les créneaux officiels`.
4. Tester `https://ev2025.github.io/pssr/inscription.html`.
5. Tester `https://ev2025.github.io/pssr/member/dashboard.html`.
6. Créer un coach seulement si nécessaire, puis tester `https://ev2025.github.io/pssr/coach/`.

## Ce qui reste Phase 2B/3

- Paiement en ligne.
- Application mobile.
- Backend FastAPI / MongoDB.
- React complet.
