# Ce qui a été repris du plan global

- Slogan : “Bouge ton corps, avance dans ta vie !”
- Tarif solidaire : 165€ / année académique
- Lieu : Gare de l’Ouest — Centre Sportif Pythagoras · urban.brussels — Molenbeek
- Collaboration : L’école du dos
- Créneaux sportifs officiels
- Ateliers socio-professionnels
- Parcours PSSR en 6 étapes : ARF, BSS, PDS, APA, CPE, SRS

- Documentation séparée dans `docs/`

## Pourquoi

Le site actuel fonctionne déjà sur GitHub Pages + Firebase. Ajouter des modules non configurés dans le code casserait ou rendrait la documentation trompeuse. La version livrée stabilise l’existant tout en conservant la vision complète pour la phase 2.
