# PSSR — version actuelle stabilisée

## Stack réel déployé

- GitHub Pages : hébergement statique du site
- HTML / CSS / JavaScript : pages publiques
- Firebase Authentication : connexion admin
- Firebase Firestore : messages, réservations, paramètres et pages
- Firebase Storage : prévu pour médias si besoin

## Fonctions déjà actives

- Site public PSSR
- Formulaire de contact vers `messages`
- Formulaire de réservation vers `reservations`
- Référence de réservation visible après envoi
- Tableau de bord admin privé
- Export CSV
- Import du contenu WordPress dans Firestore

## À ne pas confondre

Le plan React / FastAPI / MongoDB est une vision future. Il ne correspond pas au site actuellement déployé sur `https://ev2025.github.io/pssr/`.
