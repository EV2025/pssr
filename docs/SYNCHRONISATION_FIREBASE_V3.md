# Synchronisation Firebase V3

Collections prévues :

- `messages` : demandes de contact.
- `reservations` : demandes de réservation.
- `users` : clients / membres créés via inscription.
- `services` : services, tarifs et catégories.
- `slots` : calendrier / créneaux.
- `payments` : suivi manuel des paiements.
- `notifications` : notifications internes.
- `emailLogs` : traces d'emails à envoyer ou envoyés.
- `pages` : contenu importé depuis WordPress.
- `admins` et `coaches` : autorisations privées.

Important : le paiement en ligne et l'envoi email réel ne sont pas activés dans GitHub Pages seul. Ils doivent passer par un backend sécurisé ou Firebase Cloud Functions.
