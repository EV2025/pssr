# À lire avant upload GitHub

Cette version est le dossier complet stabilisé du site PSSR.

## Ce qu’il faut uploader

Dans GitHub Desktop, copie tout le contenu de ce dossier dans ton dépôt local `pssr`, puis fais :

1. Commit to main
2. Push origin

À la racine GitHub, tu dois voir :

- `index.html`
- `reservation.html`
- `merci.html`
- `roadmap.html`
- `admin/`
- `assets/`
- `data/`
- `docs/`
- `wp-content/`
- `firestore.rules`
- `storage.rules`
- `.nojekyll`

## Ce qui a été stabilisé

- Site public GitHub Pages
- Firebase déjà configuré
- Admin Firebase existant
- Messages / réservations Firestore
- Référence de réservation PSSR
- Page Roadmap
- Documentation séparée pour la phase 2
