Choix technique

Pourquoi ne pas changer le stack maintenant ?

Le site fonctionne déjà avec GitHub Pages et Firebase. Passer à React + FastAPI + MongoDB demanderait un backend séparé, un déploiement supplémentaire et une migration complète. La bonne méthode est donc de stabiliser la version actuelle, puis de traiter la grande plateforme comme une phase 2.
